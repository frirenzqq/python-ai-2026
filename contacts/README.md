# Contact Book

Простая консольная записная книжка с сохранением контактов в JSON.

## Стек
- Python 3
- json (встроенная библиотека)

## Запуск
python contacts_app.py

## Возможности
- Добавление контакта (с проверкой на дубликат)
- Поиск контакта по имени
- Просмотр всех контактов
- Автоматическое сохранение в contacts.json
